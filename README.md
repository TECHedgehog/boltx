# boltx

A TUI tool for setting up Linux systems. Built for newcomers and experienced users who want a faster path to a working machine.

## Status

Early development — not usable yet.

## Built with

- [Bubbletea](https://github.com/charmbracelet/bubbletea) — TUI framework
- [Lipgloss](https://github.com/charmbracelet/lipgloss) — terminal styling

## Roadmap

- [x] Main menu
- [ ] Use case detection (VPS vs dev machine)
- [ ] Setup categories (firewall, SSH, users, packages...)
- [ ] Per-category configuration steps
- [ ] Apply changes
